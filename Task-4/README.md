# ApexPlanet Web Development Internship — Project

A responsive multi-page website built during the ApexPlanet Software Pvt. Ltd. Web Development Internship.

**Live Demo:** https://aradhya-apexplanet-webdev.netlify.app

## Features

- Responsive 6-page website (Home, About, Services, Portfolio, Blog, Contact)
- Dark mode, hamburger menu, image slider, FAQ accordion, form validation
- **Weather App** — search weather by city (OpenWeatherMap API)
- **Movie Search App** — search movies with filters (OMDB API)
- **Tools App** — utilities and quotes
- **Expense Tracker** — add/track expenses with charts (IndexedDB + Chart.js)

## Tech Stack

HTML5, CSS3, Vanilla JavaScript, OpenWeatherMap API, OMDB API, IndexedDB, Chart.js, Netlify (hosting)

## Setup

1. Clone the repo: `git clone https://github.com/5Aradhya/web-development.git`
2. Open `Task-4/index.html` in a browser to run locally.
3. To use Weather/Movie apps, add your own free API keys in `weather.js` and `movies.js` (get keys from OpenWeatherMap and OMDB — see `.env.example`).

## Folder Structure

Task-4/
- index.html, about.html, service.html, etc. → main site pages
- weather.html, weather.js → Weather App
- movies.html, movies.js → Movie Search App
- tools.html, tools.js → Tools App
- expenses.html, expenses.js → Expense Tracker
- style.css, script.js → shared styles/scripts
